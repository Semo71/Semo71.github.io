# Osamah Fatah — Portfolio (HTML/CSS/JS)

## Run locally
Open `index.html` with a local server (recommended):

- VS Code: install "Live Server", then right click `index.html` → "Open with Live Server"

## Deploy to GitHub Pages
1. Create a repo (e.g. `portfolio`) on GitHub
2. Upload all files in this folder to the repo root
3. GitHub → Settings → Pages
4. Source: `Deploy from a branch`
5. Branch: `main` / root
6. Wait for Pages URL, then open it

## Edit content
Update: `assets/data/content.json`
- English + Arabic content is stored there.
- Projects have placeholder links — paste your App Store / Play Store / GitHub links.

## Notes
- Language toggle: EN/AR (with proper LTR/RTL)
- Theme toggle: Light/Dark (defaults to system theme)
- Contact form uses `mailto:` (no backend needed)
